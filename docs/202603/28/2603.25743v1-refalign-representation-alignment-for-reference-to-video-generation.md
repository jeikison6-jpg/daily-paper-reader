## 摘要
参考图像到视频（R2V）生成是一种可控视频合成范式，它利用文本提示和参考图像共同约束生成过程，从而支持个性化广告和虚拟试穿等应用。在实践中，现有的 R2V 方法通常在参考图像的 VAE 潜表示之外引入额外的高层语义或跨模态特征，并将它们共同输入到扩散 Transformer（DiT）中。这些辅助表示提供了语义引导并充当隐式对齐信号，可以部分缓解 VAE 潜空间中的像素级信息泄漏。然而，由于异构编码器特征之间的模态不匹配，它们可能仍难以解决“复制-粘贴”伪影和多主体混淆问题。在本文中，我们提出了 RefAlign，这是一个表示对齐框架，它显式地将 DiT 参考分支特征与视觉基础模型（VFM）的语义空间进行对齐。RefAlign 的核心是一个参考对齐损失，它拉近同一主体的参考特征与 VFM 特征以提高身份一致性，同时推开不同主体的对应特征以增强语义辨别力。这种简单而有效的策略仅在训练期间应用，不产生推理时间开销，并在文本可控性和参考保真度之间实现了更好的平衡。在 OpenS2V-Eval 基准测试上的广泛实验表明，RefAlign 在总分（TotalScore）上优于当前最先进的方法，验证了显式参考对齐在 R2V 任务中的有效性。

## Abstract
Reference-to-video (R2V) generation is a controllable video synthesis paradigm that constrains the generation process using both text prompts and reference images, enabling applications such as personalized advertising and virtual try-on. In practice, existing R2V methods typically introduce additional high-level semantic or cross-modal features alongside the VAE latent representation of the reference image and jointly feed them into the diffusion Transformer (DiT). These auxiliary representations provide semantic guidance and act as implicit alignment signals, which can partially alleviate pixel-level information leakage in the VAE latent space. However, they may still struggle to address copy--paste artifacts and multi-subject confusion caused by modality mismatch across heterogeneous encoder features. In this paper, we propose RefAlign, a representation alignment framework that explicitly aligns DiT reference-branch features to the semantic space of a visual foundation model (VFM). The core of RefAlign is a reference alignment loss that pulls the reference features and VFM features of the same subject closer to improve identity consistency, while pushing apart the corresponding features of different subjects to enhance semantic discriminability. This simple yet effective strategy is applied only during training, incurring no inference-time overhead, and achieves a better balance between text controllability and reference fidelity. Extensive experiments on the OpenS2V-Eval benchmark demonstrate that RefAlign outperforms current state-of-the-art methods in TotalScore, validating the effectiveness of explicit reference alignment for R2V tasks.

### RefAlign: 面向参考图像到视频生成的表示对齐论文总结

#### 1. 核心问题与整体含义（研究动机和背景）
参考图像到视频（Reference-to-Video, R2V）生成旨在根据文本提示和参考图像合成视频。现有的方法通常将参考图像的 VAE 潜变量与额外的语义特征（如 CLIP）共同输入扩散 Transformer（DiT）。然而，由于不同编码器特征之间的**模态不匹配（Modality Mismatch）**，现有的 R2V 模型常面临两大痛点：
*   **复制-粘贴伪影（Copy-paste Artifacts）：** 生成的视频过度重复参考图像的像素细节，缺乏动态变化。
*   **多主体混淆（Multi-subject Confusion）：** 当存在多个参考主体时，它们的特征容易相互干扰和融合。
论文的研究动机在于通过**显式的表示对齐**，利用视觉基础模型（VFM）的强语义判别力来规范 DiT 的内部特征，从而提升身份一致性和语义辨别力。

#### 2. 方法论：核心思想与关键技术
RefAlign 提出了一个简单的训练策略，其核心是**参考对齐损失（Reference Alignment Loss, RA Loss）**：
*   **核心思想：** 在训练阶段，将 DiT 内部参考分支的中间特征投影到预训练 VFM（如 DINOv3）的特征空间中。通过显式约束，使 DiT 学习到更具判别力的语义表示。
*   **关键技术细节：**
    *   **正向对齐（Positive Alignment）：** 使用余弦相似度拉近 DiT 提取的参考特征与 VFM 提取的同主体特征之间的距离。
    *   **负向对齐（Negative Alignment）：** 引入负样本对齐机制，通过一个边际损失（Margin Loss）推开不同主体之间的特征，以解决多主体混淆问题。
    *   **对齐位置：** 仅在 DiT 的前 $K$ 个块（实验设定为前 9 层）的自注意力模块中应用对齐。
    *   **推理零开销：** VFM 和投影 MLP 仅在训练时使用，推理时完全舍弃，因此不会增加生成视频时的计算负担。

#### 6. 主要结论与发现
*   **性能领先：** RefAlign-14B 在 OpenS2V-Eval 上取得了 **60.42%** 的 TotalScore，是目前公开报道中的最高分。
*   **对齐深度影响：** 适度的对齐深度（如前 9 层）能平衡身份保真度和生成的自然度；过深对齐会导致身份特征过于僵硬。
*   **编码器选择：** DINOv3 在保持身份一致性方面表现最佳，而 SigLIP2 在美学和自然度上略有优势。
*   **解决痛点：** 显式对齐显著减少了“复制-粘贴”现象，使生成的视频在遵循参考图的同时具有更合理的动态变化。

#### 7. 优点（亮点）
*   **显式约束：** 改变了以往仅靠隐式特征注入的思路，通过显式损失函数解决了模态不匹配问题。
*   **零推理成本：** 巧妙地将复杂编码器的知识在训练期“蒸馏”给 DiT，推理时无需挂载额外的 VFM 或 MLLM。
*   **多主体处理：** 负向对齐机制有效提升了模型区分多个不同参考主体的能力。

#### 8. 不足与局限
*   **数据多样性：** 论文提到目前的训练数据多样性仍有限，导致在指令遵循和参考忠实度之间仍存在权衡空间。
*   **视频长度：** 受限于 Wan2.1 基础模型，目前仅支持 81 帧视频生成，无法直接生成超长视频。
*   **引导信号单一：** 仅使用单个 VFM 可能无法捕捉视觉表示的所有维度，未来可能需要结合多个 VFM 的特征。
