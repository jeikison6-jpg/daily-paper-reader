## 摘要
身份保持视频生成为创意表达提供了强大的工具，允许用户定制包含其喜爱角色的视频。然而，现有方法通常是针对单一身份参考进行设计和优化的。这一潜在假设由于无法充分适应现实世界中多样化的输入格式，限制了创作的灵活性。依赖单一来源还构成了一个病态场景，导致了固有的歧义设置，使得模型难以在新的语境中忠实地再现身份。为了解决这些问题，我们提出了 AnyID，这是一个超高保真身份保持视频生成框架，具有两个核心贡献。首先，我们引入了一种可扩展的全参考（omni-referenced）架构，有效地将异构身份输入（如人脸、肖像和视频）统一为内聚的表示。其次，我们提出了一种主参考（primary-referenced）生成范式，该范式将一个参考指定为规范锚点，并使用一种新颖的差异提示（differential prompt）来实现精确的属性级可控性。我们在大规模、精心策划的数据集上进行训练以确保鲁棒性和高保真度，随后使用强化学习进行最终的微调阶段。该过程利用了基于人类评估构建的偏好数据集，其中标注者根据两个关键标准对视频进行成对比较：身份保真度和提示可控性。广泛的评估验证了 AnyID 在不同任务设置下均实现了超高的身份保真度以及卓越的属性级可控性。

## Abstract
Identity-preserving video generation offers powerful tools for creative expression, allowing users to customize videos featuring their beloved characters. However, prevailing methods are typically designed and optimized for a single identity reference. This underlying assumption restricts creative flexibility by inadequately accommodating diverse real-world input formats. Relying on a single source also constitutes an ill-posed scenario, causing an inherently ambiguous setting that makes it difficult for the model to faithfully reproduce an identity across novel contexts. To address these issues, we present AnyID, an ultra-fidelity identity-preservation video generation framework that features two core contributions. First, we introduce a scalable omni-referenced architecture that effectively unifies heterogeneous identity inputs (e.g., faces, portraits, and videos) into a cohesive representation. Second, we propose a primary-referenced generation paradigm, which designates one reference as a canonical anchor and uses a novel differential prompt to enable precise, attribute-level controllability. We conduct training on a large-scale, meticulously curated dataset to ensure robustness and high fidelity, and then perform a final fine-tuning stage using reinforcement learning. This process leverages a preference dataset constructed from human evaluations, where annotators performed pairwise comparisons of videos based on two key criteria: identity fidelity and prompt controllability. Extensive evaluations validate that AnyID achieves ultra-high identity fidelity as well as superior attribute-level controllability across different task settings.

*   **核心问题**：现有的身份保持（Identity-Preserving）视频生成方法大多依赖**单一参考图像**。
*   **研究动机**：
    *   **病态问题（Ill-posedness）**：仅凭一张2D静态图难以推断复杂的3D面部结构和动态表情，导致在视角切换或大幅度动作时出现身份偏移（Identity Drift）。
    *   **缺乏灵活性**：现实中用户可能拥有视频、多角度照片等多种素材，单一参考源限制了创作自由度。
    *   **属性冲突**：当提供多个参考时，不同背景、光影下的属性（发型、服装）可能产生冲突，导致模型难以确定生成基准。

### 2. 方法论
AnyID 提出了一个超高保真的通用框架，核心技术包括：
*   **全参考架构（Omni-Referenced Architecture）**：
    *   **统一表示**：不使用专门的视觉专家模型（如 CLIP/Face Encoder），而是直接利用预训练的 **VAE 编码器** 将图像和视频帧统一编码为 Latent 空间表示。
    *   **时间轴拼接**：将多个参考源（主参考+辅助参考）在时间维度上进行拼接，并赋予其 `timestep=0`（表示无噪声），与带噪的目标视频 Latent 共同输入 DiT 模型。
*   **主参考生成范式与差异化提示词（Differential Prompt）**：
    *   **视觉锚点**：指定一个“主参考”作为静态属性（发型、服装、背景）的基准。
    *   **差异化描述**：引入“差异化提示词”概念，仅描述目标视频相对于主参考的**变化部分**（如：改变表情、动作或背景），未提及的部分则默认保持一致。
    *   **提取流程**：利用 VLM 提取主参考和目标的属性，再通过 LLM 对比并剔除相似项，生成精简的差异化指令。
*   **人类中心强化学习（Human-Centric RL）**：
    *   采用 **DPO（直接偏好优化）** 算法，基于人类对“身份保真度”和“提示词可控性”的偏好数据进行微调，解决 MSE 损失函数导致的画面过平滑和细节丢失问题。

### 3. 实验设计
*   **数据集**：基于 PortraitGala 构建了大规模元数据池，包含 10 万个身份组和 30 万个视频剪辑，并人工合成了人脸、肖像、视频等多种参考形式。
*   **Benchmark**：构建了包含 50 位名人（涵盖 AI、体育、演艺等领域）的评估集，分为 IPT2V（仅保持身份）和 IEPT2V（保持身份+特定元素）两类任务。
*   **对比方法**：包括学术界和工业界的领先模型：ConsisID、FantasyID、Phantom 和 SkyReels-A2。
*   **评估指标**：
    *   **身份保真度**：Holi-Arc/Holi-Cur（计算生成脸部与所有参考脸部的平均余弦相似度）。
    *   **元素一致性**：Ele-CLIP/Ele-DINO。
    *   **可控性与质量**：利用 VLM 对外观、动作、背景及静态/动态质量进行评分。

*   **基础模型**：基于 **Wan-5B** 视频生成模型实现。
*   **算力消耗**：总计约 **4,500 NVIDIA A100 GPU 小时**。
    *   监督微调（SFT）阶段：约 4,000 小时。
    *   强化学习（RL）阶段：约 500 小时。
*   **训练细节**：采用 LoRA 方式进行微调，分辨率为 1280x704，帧数为 121 帧（约 5 秒视频）。

### 5. 实验数量与充分性
*   **实验规模**：进行了 50 组 IPT2V 和 50 组 IEPT2V 的定量评估，每组包含多个生成案例。
*   **消融实验**：针对“主参考+差异化提示词”、“辅助参考（AR）”以及“强化学习（RL）”分别做了消融对比，验证了各模块对解决属性冲突、提升动态保真度和画面质感的有效性。
*   **用户研究**：邀请 20 名参与者进行了 40 组一对一盲测，AnyID 在所有维度上的胜率均超过 60%-90%。
*   **评价**：实验设计全面，涵盖了定量指标、定性视觉对比、用户主观评价及消融分析，对比基线均为当前最先进模型，实验结果具有较高的客观性和说服力。

### 6. 主要结论与发现
*   **多参考的必要性**：引入多个参考源能显著消除单图生成的歧义，使模型能够学习到身份的 3D 结构和动态特征（如痣的位置、肌肉运动习惯）。
*   **差异化控制的优势**：通过差异化提示词，模型能更精准地识别哪些属性需要保留，哪些需要改变，有效解决了“误伤”非目标属性的问题。
*   **RL 的增益**：强化学习显著提升了皮肤纹理、发丝光泽等高频细节的还原度，使生成结果更符合人类审美。

### 7. 优点
*   **通用性强**：支持任意形式的视觉参考（图片、视频、局部裁剪），适应性极广。
*   **超高保真**：在大幅度动作和视角旋转下，身份一致性远超现有单参考模型。
*   **控制精准**：差异化提示词的设计非常巧妙，降低了用户撰写复杂 Prompt 的负担，同时提升了属性级控制的准确性。

### 8. 不足与局限
*   **流程复杂性**：差异化提示词的提取依赖于额外的 VLM 和 LLM，增加了推理流水线的复杂度和潜在的延迟。
*   **偏见与风险**：由于能实现超高保真的身份还原，该技术存在被滥用于制作深度伪造（Deepfake）视频的风险。
*   **应用限制**：虽然支持多参考，但如果所有参考源质量均较差或视角单一，提升效果可能会受限。
