Abstract 

We study reinforcement learning (RL) with linear function approximation in Markov Decision Processes (MDPs) satisfying linear Bellman completeness вҖ”a fundamental setting where the Bellman backup of any linear value function remains linear. While statistically tractable, prior computationally efficient algorithms are either limited to small action spaces or require strong oracle assumptions over the feature space. We provide a computationally efficient algorithm for linear Bellman complete MDPs with deterministic transitions , stochastic initial states, and stochastic rewards. For finite action spaces, our algorithm is end-to-end efficient; for large or infinite action spaces, we require only a standard argmax oracle over actions. Our algorithm learns an Оө-optimal policy with sample and computational complexity polynomial in the horizon, feature dimension, and 1/Оө.

# 1 Introduction 

Reinforcement Learning (RL) is a standard framework for sequential decision making (Sutton and Barto, 2018), with applications ranging from game playing (Mnih et al., 2015; Silver et al., 2016) and robotics (Andrychowicz et al., 2020) to training and fine-tuning large language models (Ouyang et al., 2022; Schulman et al., 2017). In many of these applications, the state and action spaces are large or infinite, making function approximation essential to generalize across states and actions (Bertsekas and Tsitsiklis, 1996; Sutton and Barto, 2018). A classical and widely studied approach is linear value function approximation , where one represents value functions as linear functions of pre-specified feature vectors. This approach dates back decades (Bradtke and Barto, 1996; Sutton and Barto, 2018) and continues to underpin many modern empirical methods (Mnih et al., 2015; Schulman et al., 2017). 

Linear Bellman completeness. This paper focuses on linear Bellman complete MDPs, a fundamental setting in RL with function approximation. In this setting, we are given feature maps П•h вҲ¶ X Г— A вҶ’ Rd for each step h in the horizon (where X and A are the state and action spaces, respectively), and we assume that the Bellman backup of any linear function (in these features) remains linear. The Bellman backup at step h

maps a function fh+1 вҲ¶ X Г— A вҶ’ R to the function (x, a ) вҶҰ E[rh + max aвҖІ fh+1(xh+1, a вҖІ) вҲЈ xh = x, ah = a]; it is the core operator underlying value iteration. Formally, linear Bellman completeness means that for any linear function вҹЁП•h+1(x, a ), Оё вҹ©, its Bellman backup at step h can be expressed as вҹЁП•h(x, a ), Оё вҖІвҹ© for some coefficient vector ОёвҖІ. This structural assumption is attractive for several reasons: 

вҖў Generality: Linear Bellman completeness strictly generalizes several well-studied RL models, including linear MDPs (Jin et al., 2020; Yang and Wang, 2019), and also captures the Linear Quadratic Regulator (LQR), a cornerstone model in control theory (Bertsekas and Tsitsiklis, 1996; Wu et al., 2024).

вҖў Necessity for value iteration: It is a necessary condition for the success of least-squares value iteration (LSVI), one of the most classical algorithms in RL (Bradtke and Barto, 1996). Without Bellman completeness, LSVI can diverge arbitrarily (Tsitsiklis and Van Roy, 1996). 

вҖў Statistical tractability: When the learner has access to sufficiently exploratory data, linear Bellman completeness is sufficient for statistically efficient learning (Munos, 2005; Munos and SzepesvГЎri, 2008; Zanette et al., 2020b). 

The computational challenge. Despite its simplicity and statistical tractability, linear Bellman complete-ness appears to exhibit a statistical-computational gap . While we know that polynomial sample complexity is achievable, the proposed algorithms that attain it are computationally intractable (Zanette et al., 2020b; Jin et al., 2021). This gap is particularly notable given the centrality of linear Bellman completeness to the theory of value iteration. The core difficulty is exploration : algorithms must collect data with sufficient coverage to learn a near-optimal policy. Classical approaches that rely on data from a fixed exploratory policy can fail if that policy does not provide adequate coverage of the state-action space, leading to poor sample efficiency or suboptimal policies (Munos, 2005). Overcoming this requires careful exploration, but the two principal exploration paradigms that have proven successful in simpler RL settings both break down for linear Bellman complete MDPs: 

вҖў Optimistic bonuses. In tabular MDPs and linear MDPs, algorithms such as UCBVI (Azar et al., 2017; Jin et al., 2018) and LSVI-UCB (Jin et al., 2020) add exploration bonuses to the reward that scale inversely with the frequency of visits to each state (or feature direction). This вҖңlocal optimismвҖқ approach is computationally efficient but requires that the value function class be complete with respect to these exploration bonuses. Unfortunately, for linear Bellman completeness, standard bonus functions (e.g., quadratic functions of the features) do not preserve linearity under Bellman backups, breaking this approach (Zanette et al., 2020b). 

вҖў Global optimism and version spaces. An alternative approach is to maintain a version space of plausible value functions and construct policies that are optimistic with respect to this set (Jiang et al., 2017; Zanette et al., 2020b; Jin et al., 2021; Du et al., 2021). While statistically efficient, these methods require solving complex nonconvex optimization problems at each step, making them computationally intractable even in the linear case. As a result, the following fundamental question has remained open: 

Is there an algorithm that learns a near-optimal policy in a linear Bellman complete MDP using 

poly (d, H, 1/Оө) samples and computation time? 

Even the special case where the action space is a small constant was only recently resolved (Golowich and Moitra, 2024a), but their approach has complexity exponential in вҲЈAвҲЈ. Moreover, their algorithm requires a bounded Q-function parameter assumption. As shown in (Wu et al., 2024, Section 3.1), this assumption significantly limits generality: it excludes linear MDPs as a special case, since Bellman backups can amplify parameter norms arbitrarily (we discuss this limitation in more detail in Section I). Prior to this, Zanette et al. (2020c) gave the first computationally efficient algorithm (FRANCIS), but under a strong reachability assumption requiring that every feature direction be reachable. 

Our setting and contributions. We study the setting where the state transition dynamics are deterministic 

but the initial state and rewards are stochastic (Wen and Van Roy, 2017; Du et al., 2020; Wu et al., 2024). This setting is practically relevant (e.g., in simulators, game environments, or robotic systems with known physics but stochastic rewards), and it remains theoretically challenging: the learner must still explore to estimate value functions from noisy reward feedback and handle stochastic initial states. Importantly, even this restricted setting has remained open for large action spaces. While Wu et al. (2024) studied this setting, their approach is not end-to-end computationally efficient: it relies on a linear optimization oracle over the feature space (one that solves problems of the form sup (x,a )вҲҲX Г—A П•h(x, a )вҠәОё for various Оё). Such an oracle is not needed in, e.g., the linear MDP setting, and its requirement represents a significant gap. Resolving this discrepancy is an important step toward understanding the general problem. 2Our main contribution is a computationally efficient algorithm for linear Bellman complete MDPs with deterministic transitions, large action spaces, and stochastic rewards and initial states. For finite action spaces, our algorithm is end-to-end computationally efficient and requires no oracle access. For infinite action spaces, we require only an argmax oracle over actions (i.e., argmax aвҲҲA П•h(x, a )вҠәОё for a given x and Оё), which is the standard oracle assumption used even in computationally efficient linear MDP algorithms (Jin et al., 2020). At a high level, the algorithm proceeds layer-by-layer, constructing a small set of policies that induce feature vectors spanning the relevant subspaces at each layer. The key insight is that deterministic dynamics enable efficient discovery of all relevant feature directions without requiring the linear optimization oracle that prior work relies upon. While the paradigm of first learning a policy cover and then planning is well-established (Du et al., 2019; Mhammedi et al., 2023a,b), adapting it to linear Bellman completeness requires new ideas. The novelty in our approach lies in: (i) exploiting deterministic dynamics to efficiently construct the cover without solving complex optimization problems, and (ii) reducing stochastic-reward planning to deterministic-reward planning by first estimating the instantaneous reward parametersвҖ”this reduction allows us to bypass the bounded 

Q-function parameter assumption required by prior work. 

Organization. Section 2 formalizes the setting and introduces notation. Section 3 presents a warm-up algorithm for finite action spaces, and Section 4 extends to large action spaces. The appendix contains related work (Section I), proofs of the main theorems, and supporting technical lemmas.

# 3 Warm Up: An Efficient Algorithm Under Finite Actions 

In this section, we present a simpler algorithm and analysis for finding a near-optimal policy in linear Bellman complete MDPs (under Assumptions 2.1 and 2.2) that runs in polynomial time, albeit with polynomial dependence on the number of actions A. Our approach combines existing algorithmic building blocks: the 

Optimistic Constraint Propagation (OCP ) algorithm of Wen and Van Roy (2017) and the robust barycentric spanner construction from Mhammedi et al. (2023a). 

Prior work and context. Wu et al. (2024) observed that OCP (Wen and Van Roy, 2017) extends to linear Bellman completeness when rewards and transitions are both deterministic, but noted it may not apply when rewards are stochastic. Here we show that OCP , combined with additional machinery, can tackle stochastic rewardsвҖ”with complexity polynomial in A. In Section 4, we present our main algorithm which removes dependence on A.

4High-level approach. Our approach adopts the two-phase exploration-exploitation principle used in prior work (Du et al., 2019; Misra et al., 2020; Mhammedi et al., 2023b), with Mhammedi et al. (2023a) being closest to ours: 

вҖў Phase I (Exploration): Build a policy cover layer-by-layer. A policy cover is a collection of spanning policies 

О“1вҲ¶H such that for each layer h, the expected feature vectors {EПҖ [П•h(xh, ah)] вҲ¶ ПҖ вҲҲ О“h} approximately span the space of all reachable expected features. This is achieved via barycentric spanner computation. 

вҖў Phase II (Exploitation): Estimate reward parameters via regression on rollouts from the policy cover, then use policy optimization. By estimating rewards first , we bypass the bounded Q-function parameter assumption (Section 2). Phase I uses only deterministic intrinsic rewards (linear in features by construction), for which OCP succeeds. Phase II estimates stochastic reward parameters first, reducing to a deterministic setting where OCP applies.

# 4 An Efficient Algorithm for Large Action Spaces 

In this section, we address the significantly more challenging setting where the action space A is large or infinite, making OCP computationally intractable. We assume access only to an argmax oracle over actions as described in Section 2. 

Why not use PSDP or OCP? Since OCPвҖҷs computational cost scales polynomially with вҲЈAвҲЈ (see Section 3), we must consider alternative approaches. A natural idea is to use policy optimization algorithms such as PSDP to build the policy cover layer-by-layer. However, PSDP requires bounded Q-function parameters to obtain tractable guaranteesвҖ”an assumption that may exclude even linear MDPs as remarked in Section 2. Furthermore, algorithms like PSDP require roll-in policies with good state coverage. One could imagine using the approach of Algorithm 2: building approximate spanners layer-by-layer and feeding the spanner from previous layers to the PSDP routine. This is, for example, the approach taken by Mhammedi et al. (2023a) in the low-rank MDP setting. However, in the linear Bellman complete setting, coverage errors compound across layers: if the spanner at layer h fails to span some relevant feature directions, these gaps propagate and amplify at subsequent layers. Since RobustSpanner can only compute approximate spanners, this compounding is unavoidable. Thus, we need a fundamentally different approach. 

Our approach: FQI with exact linear optimization. Instead of using OCP or PSDP, we use Fitted Q-Iteration ( FQI ) (Munos and SzepesvГЎri, 2008) and perform linear optimization exactly within each layer. # 5 Conclusion 

We have presented computationally efficient algorithms for reinforcement learning in linear Bellman complete MDPs with deterministic transitions. This brings us a step closer to resolving the general case with stochastic transitions, which remains an important open problem. Our techniques may also be applicable to large language model (LLM) fine-tuning via reinforcement learning. In this setting, the вҖңenvironmentвҖқ is a sequence of tokens, and the transition dynamics are deterministic: given a prompt and a sequence of generated tokens, the next state (the concatenated sequence) is fully determined. A natural setting to investigate is the linear softmax parameterization, studied recently by Foster et al. (2025); Xiong et al. (2023); Cen et al. (2024). This parameterization arises naturally when fine-tuning only the final layer of a pretrained autoregressive language model such as GPT-2 (Radford et al., 2019), while keeping all earlier layers fixed. Due to the linearity of the policy class and the determinism of the transitions, this setting shares key structural similarities with ours, making it a promising direction for future work. 11 Acknowledgments

# Auxiliary Results and Additional Related Work 

# I Related Works 

Tabular and linear MDPs. The simplest RL setting is the tabular case, where both the state and action spaces are finite and small enough that every state-action pair can be visited multiple times. Here, computationally efficient and near-optimal algorithms have been known for decades, including classical methods such as Q-learning (Watkins and Dayan, 1992) and value iteration (Bertsekas and Tsitsiklis, 1996), as well as modern optimistic exploration algorithms such as UCBVI (Azar et al., 2017), Q-learning-UCB (Jin et al., 2018), and variants achieving optimal sample complexity (Zhang et al., 2020). Statistically efficient approaches. Linear Bellman completeness strictly generalizes linear MDPs but does not impose any structure on the transition functionвҖ”it only requires that Bellman backups of linear value functions remain linear (Zanette et al., 2020b). This makes the setting significantly more challenging computationally, since techniques that work for linear MDPs (e.g., adding quadratic exploration bonuses) no longer apply: quadratic functions do not remain linear under Bellman backups in this setting. Early work established that linear Bellman completeness is statistically tractable. Munos (2005); Munos and SzepesvГЎri (2008) showed that with sufficiently exploratory data, fitted Q-iteration succeeds under Bellman completeness. Zanette et al. (2020b) proposed ELEANOR, an algorithm that achieves polynomial sample complexity in the online setting by maintaining a version space and performing global optimistic planning. Similarly, Jin et al. (2021) proposed GOLF using Bellman eluder dimension, and Du et al. (2021) studied bilinear classes. However, all these algorithms are computationally intractable : they require solving nonconvex optimization problems to implement optimistic planning over complex version spaces, even when the value function class is linear. 

Reachability and constant actions. The first computationally efficient algorithm for linear Bellman completeness was FRANCIS (Zanette et al., 2020c), which works under a strong reachability assumption that for every direction in feature space, there exists a policy whose expected features have non-trivial projection onto that direction. The question of computationally efficient learning without reachability remained open until very recently. Golowich and Moitra (2024a) made a breakthrough by introducing a 

local optimism approach that constructs exploration bonuses preserving Bellman linearity. This was the first polynomial-time algorithm for the general setting. However, their approach has a critical limitation: the 35 sample and computational complexity scale as poly (d, H, 1/Оө)O(вҲЈ AвҲЈ) , exponential in the number of actions. As they note, this restricts their method to settings where вҲЈAвҲЈ is a small constant (e.g., вҲЈAвҲЈ = 2 or 3). Additionally, their algorithm requires a bounded Q-function parameter assumption: that the parameter Оё satisfying 

Qh(вӢ…, вӢ…) = П•h(вӢ…, вӢ…)вҠәОё has вҲҘОёвҲҘ2 вүӨ R for some polynomial R in d. As shown by Wu et al. (2024), this assumption limits generality: it excludes linear MDPs as a special case, since Bellman backups can amplify parameter norms arbitrarily in that setting. 

Deterministic dynamics setting. The special case of deterministic transition dynamics has been studied as a stepping stone toward the general problem, and is practically relevant for applications such as game environments and robotic systems with known kinematics. Wen and Van Roy (2017) provided a polynomial-time algorithm when both transitions and rewards are deterministic. For finite action spaces, our algorithm builds on their work, using their subroutines for constructing approximate spanners and for policy optimization (see Section 3). For large action spaces, we develop a different approach based on exact identification of relevant feature directions (see Section 4). Du et al. (2020) extended this to stochastic rewards under a 

positive suboptimality gap assumption, meaning that suboptimal policies have value strictly bounded away from optimal. Most recently, Wu et al. (2024) studied the same setting as ours: deterministic transitions with stochastic rewards and initial states, without assuming a suboptimality gap. Their algorithm uses a randomization-based approach called вҖңnull space randomization,вҖқ which adds exploration noise only in the null space of previously collected data. However, their algorithm requires solving constrained regression problems that rely on a linear optimization oracle over the feature space, specifically the ability to compute 

sup (x,a )вҲҲX Г—A П•h(x, a )вҠәОё for any Оё. Our work removes this oracle assumption for finite action spaces, achieving end-to-end computational efficiency using only standard regression and value function evaluation. For infinite action spaces, we require only an argmax oracle over actions (i.e., argmax aвҲҲA П•h(x, a )вҠәОё for a given x and Оё), which optimizes only over actions rather than state-action pairs. This is a strictly weaker requirement and is the standard oracle assumption in computationally efficient algorithms for linear MDPs (Jin et al., 2020; Yang and Wang, 2019). 

Policy covers and spanning arguments. Our algorithm is inspired by the policy cover framework (Du et al., 2019; Misra et al., 2020), which constructs a small set of policies that collectively вҖңcoverвҖқ the state space. This approach has been successful in related problems such as representation learning in RL (Mhammedi et al., 2023b,a; Golowich and Moitra, 2023) and learning in POMDPs (Golowich and Moitra, 2022). A key challenge in applying policy covers to linear Bellman completeness is that coverage errors compound across layers: if the policy cover at layer h fails to span some relevant feature directions, these gaps propagate and amplify at subsequent layers, causing the cover to miss increasingly more of the feature space as the horizon grows. Our contribution is showing how deterministic dynamics help address this challenge. For finite action spaces, we show that approximate spanners suffice (see Section 3). For large action spaces, we develop machinery ( SubspaceCover ) that exactly identifies all relevant feature directions at each layer, avoiding the error compounding entirely (see Section 4). 

Exploration via randomization. Randomization has proven to be a powerful alternative to bonus-based exploration in RL. A prominent example is Randomized Least-Squares Value Iteration (RLSVI) (Osband et al., 2016), which adds Gaussian noise to least-squares estimates and has been shown to achieve near-optimal worst-case regret for linear MDPs (Agrawal et al., 2021; Zanette et al., 2020a). Ishfaq et al. (2021) developed randomization algorithms for general function approximation under bounded eluder dimension and Bellman completeness. In preference-based RL, randomization has led to the first computationally efficient algorithm with near-optimal regret for linear MDPs (Wu et al., 2023). However, these randomization approaches face challenges when applied to linear Bellman completeness: many inject noise larger than the estimation error to ensure optimism, which can cause exponential growth of parameter values over the horizon. Truncating these values works for low-rank MDPs but is problematic under linear Bellman completeness, since truncation may break the linearity property. Our work and the concurrent work of Wu et al. (2024) address this by restricting noise to the null space of collected data, though we differ in our approach to ensuring computational efficiency. 36 Broader structural conditions. Linear Bellman completeness is captured by several broader structural conditions studied in the RL literature, including Bellman rank (Jiang et al., 2017), witness rank (Sun et al., 2019), Bellman eluder dimension (Jin et al., 2021), the decision-estimation coefficient (Foster et al., 2021), and bilinear classes (Du et al., 2021). These frameworks provide different lenses for understanding when RL is statistically tractable. While statistically efficient algorithms exist for these settings, computationally efficient algorithms remain unknown in general. 

Offline setting In the offline setting, Golowich and Moitra (2024b) recently established a computationally efficient algorithm for linear Bellman completeness, leveraging different techniques suitable for the batch setting.
