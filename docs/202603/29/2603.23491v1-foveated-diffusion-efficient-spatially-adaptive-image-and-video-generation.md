## 摘要
扩散模型和流匹配模型为创意内容创作解锁了前所未有的能力，例如交互式图像和流式视频生成。然而，对更高分辨率、帧率和上下文长度日益增长的需求，使得高效生成变得越来越具有挑战性，因为计算复杂度随生成的 token 数量呈二次方增长。我们的工作旨在优化用户注视位置已知或可被估计（例如通过眼动追踪）的场景下的生成过程效率。在这些场景中，我们利用了人类视觉中依赖于离心率的敏锐度：虽然用户在注视位置周围的一个小区域（中央凹区域）内感知到极高分辨率的视觉信息，但在视野外周，分辨细节的能力会迅速下降。我们的方法首先使用一个模拟注视点分辨率的掩码来非均匀地分配 token，为中央凹区域分配较高的 token 密度，为外周区域分配较低的密度。图像或视频在混合分辨率的 token 设置下生成，产生的结果在感知上与全分辨率生成无异，同时大幅减少了 token 数量和生成时间。为此，我们开发了一种从高分辨率数据直接构建混合分辨率 token 的原则性机制，允许注视点扩散模型从现有的基础模型进行后训练，同时保持跨分辨率的内容一致性。我们通过广泛的分析和精心设计的用户研究验证了我们的方法，证明了注视点化作为高效生成的实用且可扩展维度的有效性。

## Abstract
Diffusion and flow matching models have unlocked unprecedented capabilities for creative content creation, such as interactive image and streaming video generation. The growing demand for higher resolutions, frame rates, and context lengths, however, makes efficient generation increasingly challenging, as computational complexity grows quadratically with the number of generated tokens. Our work seeks to optimize the efficiency of the generation process in settings where the user's gaze location is known or can be estimated, for example, by using eye tracking. In these settings, we leverage the eccentricity-dependent acuity of human vision: while a user perceives very high-resolution visual information in a small region around their gaze location (the foveal region), the ability to resolve detail quickly degrades in the periphery of the visual field. Our approach starts with a mask modeling the foveated resolution to allocate tokens non-uniformly, assigning higher token density to foveal regions and lower density to peripheral regions. An image or video is generated in a mixed-resolution token setting, yielding results perceptually indistinguishable from full-resolution generation, while drastically reducing the token count and generation time. To this end, we develop a principled mechanism for constructing mixed-resolution tokens directly from high-resolution data, allowing a foveated diffusion model to be post-trained from an existing base model while maintaining content consistency across resolutions. We validate our approach through extensive analysis and a carefully designed user study, demonstrating the efficacy of foveation as a practical and scalable axis for efficient generation.

### 论文总结：Foveated Diffusion（注视点扩散模型）

#### 1. 核心问题与整体含义
*   **研究动机**：现代扩散模型（如 DiT）的计算复杂度随 token 数量呈二次方增长。随着用户对高分辨率图像和长视频需求的增加，生成速度成为瓶颈。
*   **核心思想**：借鉴人类视觉系统的“注视点（Foveation）”特性——人类仅在视网膜中央凹区域（注视点）拥有极高分辨率，而外周区域的分辨率随偏心率增加而迅速下降。
*   **整体含义**：通过在注视区域分配高密度 token，在边缘区域分配低密度 token，实现空间自适应的非均匀计算分配，从而在不损失感知质量的前提下显著提升生成效率。

#### 2. 方法论
*   **核心思想**：将图像/视频划分为混合分辨率的 token 序列进行处理。
*   **关键技术细节**：
    *   **注视点 Token 化（Foveated Tokenization）**：根据注视点掩码 $M$，将图像分为高分辨率区域（保留原始 token）和外周区域（将 2x2 的 token 块下采样为 1 个 token），大幅减少总序列长度 $L$。
    *   **混合分辨率 RoPE（相位对齐旋转位置嵌入）**：针对非均匀网格修改 RoPE 索引，确保高分辨率查询（Query）能正确关注到低分辨率键（Key），维持空间一致性。
    *   **注视点后训练（Foveated Training）**：发现直接在预训练模型上进行混合分辨率推理（Naive 方式）会导致结构不一致和伪影。因此，采用 LoRA 对预训练模型进行微调。训练目标是让模型预测从高/低分辨率混合构建的“清洁注视点目标”的流速度（Flow Matching）。
    *   **推理流程**：在缩减后的 token 空间采样噪声 $\rightarrow$ 迭代去噪 $\rightarrow$ 将结果拆分为高/低分辨率部分 $\rightarrow$ 分别通过 VAE 解码 $\rightarrow$ 根据掩码进行 Alpha 混合。

*   **Benchmark 与对比方法**：
    *   **对比方法**：全分辨率生成（Full high-res）、朴素混合分辨率生成（Naive mixed-res，无训练）。

*   **实验规模**：生成了 10,000 张测试图像和 200 个测试视频进行定量评估。用户研究包含 11 名受试者，共 660 次试验。
*   **充分性**：实验涵盖了随机掩码、显著性引导掩码、边界框掩码等多种策略，验证了模型在不同交互场景下的泛化能力。
*   **客观性**：通过定量指标、定性视觉对比以及符合心理物理学标准的受试者实验，多维度证明了方法的有效性，实验设计较为严谨。

#### 6. 主要结论与发现
*   **性能飞跃**：在保持感知质量几乎不变的情况下，视频生成速度提升高达 4 倍。
*   **感知等效性**：用户研究显示，受试者在注视点观察条件下，无法区分本方法生成的内容与全分辨率内容（偏好率接近 50%）。
*   **训练的必要性**：后训练（Post-training）对于消除跨分辨率边界的结构伪影和比例失调至关重要。
*   **应用潜力**：该方法可扩展至显著性引导生成，自动将计算资源集中在画面中的关键物体（如机器人手臂、游戏中的武器）。

#### 7. 优点
*   **感知驱动**：直接针对人类视觉特性优化，而非盲目追求全图像素级精确，思路具有启发性。
*   **显著加速**：在不改变模型核心架构的前提下，通过 token 缩减实现了量级上的加速。
*   **兼容性强**：作为一种后训练方案，可以轻松适配现有的各种基于 Transformer 的扩散模型。

#### 8. 不足与局限
*   **边界伪影**：在注视点边缘偶尔会出现颜色不连续或轻微伪影，这主要是由于 VAE 分别解码后再混合导致的。
*   **VAE 限制**：目前的 VAE 并非为混合分辨率设计，未来需要开发能直接处理非均匀 token 的 VAE。
*   **多级嵌套**：目前仅实现了两级分辨率（中央 vs 外周），尚未探索更复杂的多级注视点金字塔。
*   **实时性集成**：虽然推理加速显著，但尚未在集成真实眼动仪的闭环实时流式系统中进行实测。
